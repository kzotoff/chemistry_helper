Резервное копирование

Общее описание
  • Модуль предназначен для резервного копирования пользовательских данных — страниц, базы данных, основного шаблона.
  • На странице администрирования можно найти список ранее сделанных копий и форма для создания новой копии.
  • Также здесь находится форма для создания новой копии
  • Ранее сделанную копию можно скачать (сохранить на свой компьютер), восстановить или, наконец, просто удалить.
  • При восстановлении из архива распаковываются все имеющиеся в нем файлы, файлы перезаписываются без предупреждения, будьте аккуратны! По этой же причине загрузка каких-либо копий с компьютера на сайт не предусмотрена.
  • Для создания новой резервной копии нужно:
    • ввести имя для файла (по умолчанию используется текущая дата в формате ГГГГММДДЧЧММСС), без расширения
    • ввести комментарий к архиву (не обязательно). Комментарий будет виден в поле "описание" списка имеющихся копий
    • нажать на кнопку Создать

Описание макросов

Этот модуль не распознает никаких макросов
