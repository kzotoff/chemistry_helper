Модуль для работы с БД

---------------------------------------------------------------------------------------------------
версия 0.01
версия описания 0.01
---------------------------------------------------------------------------------------------------

* Состоит из двух частей - UI и API
	* UI: всё для отображения БД и связанных с этим вещей.
		* Только вывод информации, только для пользователя (если получение сырых данных - это в API)
		* Данные из БД в различной форме (таблицы, графики, разное)
		* Контекстное меню
		* Диалоги создания, редактирования, удаления, всего такого
	* API: работа в базе - изменение информации в базе
		* Никакого специального юзерского выхлопа - только результат операции
		* Унифицированный вход:
			* метод, который надо выполнить
			* идентификатор отчета
			* идентификатор строки (опционально, например, не нужен для получения количества записей)
			* название поля
			* новое значение для поля, если требуется
	* Унифицированный выход обоих частей:
		* три специальных заголовка:
			* X-JuliaCMS-Result-Status - результат операции.
				* "OK": успешно
				* другое значение: неудача
			* X-JuliaCMS-Result-Type: тип ответа:
				* plain: простой текст
				* html: HTML-разметка для вывода пользователю
				* command: имеется команда для выполнения скриптом на стороне клиента
				* json: данные в формате JSON
			* X-JuliaCMS-Result-Command: команда для скрипта
				* reload: необходимо перезагрузить страницу (например, данные изменились)
		* в теле ответа содержится собственно ответ сервера в указанном типе.
				
				
* Архитектурно состоит из трех частей:
	* Общая входная часть
		* Сюда же входит описание действий
	* Класс для UI
	* Класс для API
