Управление меню

Общее описание

Модуль для работы с меню-подобными структурами: собственно меню в классическом виде, древовидные, каталоги и все такое.

Каждый пункт меню может ссылаться на одну из страниц сайта или вообще на произвольную страницу в интернете.

Быстрый старт
  • Первое, с чего нужно начать — создать "ведущий" элемент. На него мы будем ссылаться при выводе нужного нам меню. Формально этот элемент никак не отличается от остальных и создается так же. Нажимаем на кнопку , заполняем форму и сохраняем. Обязательным является только "название", остальные поля подробно описаны в разделе "свойства элемента меню".
    • После того, как мы создали главный элемент, создаем все пункты нашего меню, не забываем указывать наш главный элемент как родительский.
  • Самое время поместить меню на сайт. Для этого на какой-нибудь странице или даже на основном шаблоне помещаем такой макрос: 
    [macro type="menu" start-from="0" /]
    В макросе обязательно наличие элемента start-from, а значением его является идентификатор элемента, с которого нужно начать вывод - его можно посмотреть в заголовке формы редактирования.
Свойства элемента меню

Обязательным является только свойство "название"

СтраницаМожно выбрать, что элемент будет ссылаться на одну из имеющихся страниц сайта (или на меню-каталог)
Ссылка Можно просто написать ссылку куда угодно — на страницу сайта, файл в хранилище или на какой-то другой сайт.
 Если делаем ссылку на другой сайт — обязательно использовать префикс http://, иначе браузер будет пытаться открыть страницу на текущем сайте.

Меню-каталог

Кроме вывода обычного меню, у модуля есть еще один режим — каталог. Работает он следующим образом:

  1. Например, у нас есть элемент меню с псевдонимом "catalog"
  2. И в адресной строке набрано что-то вроде http::/out-site-name.com/catalog
  3. Тогда модуль меню вместо страницы с псевдонимим catalog выведет все подпункты нашего элемента с таким псевдонимом.
  4. Кроме того, каждый выведенный элемент будет содержать ссылку на следующие уровни, что позволяет перемещаться по всему меню, как по папкам.
Навигатор

Навигатор просто показывает путь к текущей странички от самого верхнего пункта меню, выглядит примерно так:

каталог > инструмент > мелкий > паяльники

Он может добавляться на странички-каталоги автоматически, а может быть вставлен вручную.

Руководство по макросам

Стандартный режим — будет выведено обычное древовидное меню, начиная с пункта "begin": 

[macro type="menu" start-from="begin" /]

Стандартный режим, с выбором шаблона структуры (о шаблонах в "технических вопросах"): 

[macro type="menu" start-from="begin" menu-template="custom"/]

Режим каталога (на самом деле, это шаблон для модуля управления содержимым, но модуль меню умеет распознавать в нем некоторые параметры). 

[macro type="content" /]

В предыдущем примере навигатор появится автоматически. Чтобы отключить его, добавим такой параметр: 

[macro type="content" show-navigator="no" /]

Вот так можно выбрать шаблон разметки для элементов каталога: 

[macro type="content" catalog-template="cool" /]

Вот так вызывается навигатор. По умолчанию будет показан путь от текущей страницы, если она (то есть, ее псевдоним) есть где-то в меню. 

[macro type="navigator" /]

А вот так можно изменить шаблон для навигатора (Этот же параметр можно использовать в макросе содержимого, когда type="content"): 

[macro type="navigator" navigator-template="navi" /]

Не показывать корневой элемент (он часто используется просто как стартовая точка, не неся никакой информации), этот же параметр можно использовать в макросе содержимого для режима каталога: 

[macro type="navigator" navigator-no-root="yes" /]

Можно выводить не весь путь до начального элемента, а последние несколько пунктов (в этом примере будут показаны только последние 3 части "пути"): 

[macro type="navigator" navigator-count="3" /]
