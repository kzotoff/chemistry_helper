-------------------------------------------------------------------------------
                     SEARCH : модуль для поиска по тексту                     
-------------------------------------------------------------------------------

Краткое описание:
	Модуль предоставляет возможность поиска по контенту и новостям (при наличии
	сооветствующих модулей)


Использование:
	Вызов модуля осуществляется с помощью GET-запроса вида:
		site.com/?module=search&search=SEARCH_STRING
	
	Проще всего разместить где-нибудь на странице поисковую форму вида
		---8<--------------------------------------------------------
		<form action="search" method="get">
			<input type="hidden" name="module" value="search" />
			<input type="text" name="search" />
			<input type="submit" value="Искать" />
		</form>
		---8<--------------------------------------------------------
		
	Определив, что его вызвали, модуль производит поиск по статичным
	пользовательским страницам (генерируемые не рассматриваются) и текстам
	новостей, полученные результаты выводит вместо контента странички,
	найденный текст оборачивается в заданный тег (например, в <b>)


Настройка:
	По состоянию на версию 1.0, только через настроечный файл. HTML-разметка
	вывода задана в файле output.xsl

	
API:
	Нет

	
Дополнительные public-функции в модуле:
	Нет

	
Управление с помошью шаблонов:
	Нет

	
Константы и дополнительные файлы:
	output.xsl : файл для трансформации XML с результатами поиска в то, что
	             будет показано на страничке


Описание настроек:

	<config>
		
		<!--
			количество символов, которые будут показаны рядом с найденным
			фрагментом, т.е., если ищем "тест" в тексте "много знаков с начала
			теста и много потом еще будет", и параметр равен "7", то поисковик
			выдаст строку:
				"...начала <b>тест</b>а и мно..."
		-->
		<chars_to_include>60</chars_to_include>
		
		<!--
			тег для оборачивания найденного текста. По умолчанию "b".
			можно использовать, например, классы (span class="highlight"),
			тогда пример выше станет таким:
				"...начала <span class="highlight">тест</span>а и мно..."
		-->
		<wrap_tag>b</wrap_tag>

	</config>

	
Полезные советы:


Известные баги:
	если искомая строка в тексте встречается несколько раз, подсвечивается
	только первое вхождение
