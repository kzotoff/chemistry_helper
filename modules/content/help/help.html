Управление страницами

Общее описание

Здесь вы можете управлять самым главным, что есть на вашем сайте — его страницами.

Если вы еще не сделали этого, закройте справку, нажмите на кнопку pages на административной панели, и вызовите справку снова.

Создание страницы

Просто нажмите кнопку  ), заполните предложенную форму (обязательно только поле "Псевдоним", подробнее в разделе описание свойств страницы), и нажмите .

Редактирование содержимого страницы
  • Если вы находитесь в разделе управления страницами, то наведите курсор мыши на нужную страницу и нажмите на эту кнопку:
  • Если вы просматриваете сайт, то для редактирования текущей страницы просто нажмите edit this page на административной панели.
  • Редактирование страницы происходит с помощью визуального редактора, напоминающего обычный текстовый редактор из обычного офисного пакета. Инструкция по работе с ним находится чуть ниже, в разделе "Визуальный редактор страниц"
  • Когда изменения внесены, нажмите кнопку save page на административной панели. Если вы передумали и сохранять страницу не надо, там же есть кнопка cancel.
Работа со списком страниц

Для изменения свойств страницы наведите курсор мыши на нужную страницу и нажмите на кнопку свойств (  ) или просто на любое место строки, где нет других кнопок, сделайте необходимые изменения и нажмите

Для редактирования содержимого страницы используйте кнопку "переход к странице" (  ).

Для удаления страницы служит кнопка удаления (  ). Страница не удаляется безвозвратно, а попадает в "корзину". Оттуда ее можно перенести обратно с помощью файлового менеджера.

Описание свойств страницы

Обязательным для заполнения являкется только поле "псевдоним"

Свойство Описание
Заголовок Текст, который будет отображен в заголовке окна при выборе этой страницы
Псевдоним
Имя страницы, как оно появляется в адресной строке. На рисунке справа это будет "informer". Допускаются большие и маленькие латинские буквы, цифры, дефис и знак подчеркивания.
Имя файла Имя файла, в котором будет храниться страница. По умолчанию будет использован псевдоним и расширение .html
meta метатеги для страницы. О формате записи подробнее написано в технических вопросах.
CSS Таблица стилей для страницы, должна находиться в папке CSS пользовательских файлов. Подробнее в документации по файловому менеджеру.
Скрипт Скрипт на языке javascript, из пользовательской папки со скриптами.
PHP-генератор Если не хочется делать статическую страницу, можно сгенерировать ее с помощью функции на языке PHP. Подробнее в "технических вопросах".
XSL Кроме всего этого, страница может представлять из себя XML документ, к которому можно применить XSL-трансформацию. Подобно таблицам стилей и скриптам, файл с трансформацией должен лежать в соответственной папке.
Технические вопросы

В шаблоне сайта за положение контента отвечает макрос 

[macro type="content" /]
Никаких дополнительных параметров не предусмотрено.

Формат метатегов:

  • Самый простой вариант — просто написать какие-то ключевые слова (то есть, буквы, цифры, точка, запятая). В таком случае к странице будет добавлена такая конструкция: 
    <meta keywords="тут-будут-ключевые-слова" />
  • Формат посложнее: 
    (атрибут|значение атрибута|содержимое) (...) (...)
    На каждую пару скобок будет создан метатег вида 
    <meta атрибут="значение атрибута" content="содержимое" />
    или, если параметр "содержимое" пустой, то 
    <meta атрибут="значение атрибута" />
  • Третий способ — просто написать метатеги, как они должны быть на странице.

О PHP-генераторе. Тут все просто. Пишем произвольную функцию, которая возвращает содержимое в виде строки. Функция должна находиться в глобальном пространстве имен (то есть, не должна содержаться внутри какого-либо класса, другой функции или где-то еще). Файл с функцией подключаем к /index.php с помощью include/include_once, вписываем название функции в поле "PHP-генератор". Теперь вместо содержимого страницы будет выводиться то, что выдает наша функция.
