Проблемы внешнего характера

Система не отслеживает свободное место на диске, поэтому его необходимо периодически проверять самостоятельно.

Если у вас есть ограничено свободное место в базе данных, не забывайте и его проверять.

В ряде случаев системе нужны права на запись в определенные папки на диске, а именно:

  • Если вы используете SQLite в качестве БД, необходим доступ на запись в файл БД. В стандартной поставке это userfiles/_data_common/cms.sqlite.
  • Кроме того, SQLite используется плагином minigallery текстового редактора, файл с базой лежит вот здесь: userfiles/_data_common/minigallery.sqlite. Сами картинки по умолчанию хранятся в userfiles/images/gallery/
  • Если используется модуль "db", то ему потребуется папка для хранения файлов, приложенных к комментариям, с правом записи в нее: userfiles/_data_modules/db/attached/
  • Наконец, модулю администрирования для сохранения параметров модулей и самой системы потребуется запись в файлы userfiles/_data_modules/*****/config.xml и в файл userfiles/_data_common/conf.php.
